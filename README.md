## Installation

**1.clone Repository & Install Packages**
```sh
git clone https://github.com/Sahal-P/fero.ai.git

cd fero.ai


```
**2.create .env file from .env.example**
```sh
SECRET_KEY=

DATABASE_NAME=
DATABASE_USER=
DATABASE_PASS=
```
**3. Run Test**
```sh
python manage.py test

```
**3. Start Server**
```sh
docker-compose up --build

```
